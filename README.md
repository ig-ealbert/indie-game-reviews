# Indie Game Reviews

## What

I want to provide more context around various indie (independent publisher) games that I've been playing. This is a collection of game reviews that provide information about the game, feedback on it, and screenshots so that others can clearly understand the game concept.

## Why

I want to talk about games, but people don't understand what I'm talking about because either they're not into games, or they have never heard of this game because it's not made by a huge studio with an enormous marketing budget. Giving them something visual will help overcome this barrier.

I also want to show off my writing skills and gaming experience, as it's good to have it in a shareable form.

I am not being paid by any of the parties involved in making or publishing these games. These reviews are my own unbiased opinions. I'm just doing this because I like them, and want to share my enjoyment of these games.

## Where

Check the `reviews` folder. I will give each one a separate file, with the name matching the name of the game. If you view it in a browser, it should render the markdown as HTML and you can read it like a webpage.
